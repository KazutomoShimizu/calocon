## 就職Termの技術

## カリキュラム外の技術
• Simple Calendar

## 実行手順

```
$ git clone git@github.com:KazutomoShimizu/calocon.git
$ cd calocon
$ bundle install
$ rails db:create && rails db:migrate
$ rails s

```
